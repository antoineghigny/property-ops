# Property-Ops Skill

This skill enables deep residential property analysis and feasibility audits.

## Procedural Rules
1. **Pre-Flight Interview:** Always ask for Intent (Home/Invest), Strategy (Classic/Co-living), and Leverage (Cash/Pledging) before research.
2. **Investigation Phase:** Execute multiple `web_search` calls for local employment hubs and market data.
3. **Execution:** Run `npm run screen` only after strategy is defined.
4. **Reporting:** Generate a 1000+ words "Expert Property Audit" with a Decision Dashboard.

## Financial Intelligence
- **Pledging Logic:** Model diversified securities at 50-60% LTV.
- **DTI Calculation:** Add 80% of rent to net income for investment cases.
- **Regional Duties:** 3% for primary residence (Wallonia 2025), 12.5% for investment.

## Commands
- `/property-ops {url}`: Full audit.
- `/property-ops-bank-package`: Professional PDF generation.
- `/property-ops-negotiate`: Price reduction strategy.
- `/property-ops-visit`: Technical guide.
