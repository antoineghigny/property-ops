# Shared Modes Context

## Core Analysis Mandates
- **Depth Over Speed:** Never render a verdict based on surface data. Perform multiple parallel searches for market context, rental yields, and urban planning.
- **Dual Perspective:** Always analyze the property from two angles:
  1. **Residence (Home):** 3% registration duty (Wallonia 2025 rule), no rental income.
  2. **Investment (Yield):** 12.5% registration duty, +80% rental income recognition.
- **Financial Precision:** Correcty distinguish between mobilizable cash and securities. If total assets cover the project, it is financeable via liquidation/pledging.
- **No data invention:** Never invent missing evidence.
- **Source Transparency:** Always cite sources and surface `manual_review_required` for critical uncertainties.
